## makefile setup

for __`both backend and frontend`__

``` shell
make setup
make run
```

for `frontend only` | port - 7000

``` shell
make FSetup
make FRun
``` 

for `backend only` | port - 8000

``` shell
make BSetup
make BRun
```
for `database migerations`

``` shell
make db-mig
```
